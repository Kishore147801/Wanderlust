**Wanderlust – Full-Stack Vacation Rental Platform**
• Built a full-stack vacation rental platform using Node.js, Express.js, and MongoDB, following the MVC
architecture for scalable application design.
• Developed RESTful APIs and implemented secure authentication and role-based authorization with
Passport.js to manage listings and user reviews.
• Integrated Mapbox API for geospatial property visualization and delivered a responsive UI using EJS and
Bootstrap.
Technologies: Node.js, Express.js, MongoDB, Passport.js, EJS, Bootstrap, CSS, Mapbox API
